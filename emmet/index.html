HTML

Ajouter un élément enfant: ">"

li>p
ul>li>p

Résultat :

Multiplier un élément : "*"

ul>li*3>p+img

Resultat :

Ajouter une class : "."

ul>wrapper

.container

Grouper des éléments multiples : "()"

ul>(header>ul>li*2>a)+footer>p
ul>(li>p+img+a)+li*2>p+img

Resultat :

Insérer du text dans un élément : "{}"

a{click me}
p>{click me}+a{here}+{to continue}

Resultat:

Ajouter une numérotation : "$"

ul>li.item$5}

Resultat:

commencer la numerotation au chiffre voulu : "$@x"

ul>li.item$@3*5

Resultat:

numéroter dans l'ordre décroissant: "$@-"

ul>li.item$@-{pizza $@-}*5

Resultat:

Remonter d'un élément enfant: "^"

div>p>em{hello}^a

Resultat:

Ajouter un attribut: "[attr_name="valeur]"

img[src=https://picsum.photos/seed/picsum/200/300]

Resultat:

En bref : "^"

div.list>ul>(li.item$>p{paragraph $}+img[src="https://picsum.photos/seed/picsum/200/300"])+li.item$@2*4>p{paragraph $@2}+img[src=""http://placesheen.com/$@1$$/$@5$$"]+a^^^footer>p{place cage vrs palcesheen}

Resultat:

le premier regroupement contient un <li> qui a 2 enfants: <p> qui contiennent du texte et une <img> qui a un attribut src

le regroupement suivant contient les <li> 2 à 5, en multipliant des liste dont les noms de class, contienu de paragraphe et valeurs d'attributs s'indrémentent selon des symtaxes differentes. chaque <li> contient ausi une ancre, invariable.

enfin les 3 $ ^^^ permettent de remonter les éléments parents pour placer le <footer> au même niveau que la <div> (sortirde <li> puis <ul> puis de <div> pour donner à ce dernier un <footer>"frère")
