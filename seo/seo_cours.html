SEO TEST

L'optimisation pour les moteurs de recherche,ou SEO (Pour Search engine optimisation), est un ensemble de technique pour optimiser la visibilité d'une page web dans les pagede résultats de recherche (SERP), pour Serch en engine result pages).

L'objectif est d'ameliorer le positionemment d'une page web dans les pages de résultats de recherche sur des sites mots-clés (selon les thème principaux du site). On considère que le positionemment d'un site est bon lorsqu'il est positionné (classé) dans la première réponses naturelles d'une recherche sur des mots-clés correspondant précisément à sa thématique.
